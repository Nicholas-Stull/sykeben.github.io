Getting Started with GitHub Pages

 1/3
Introduction

GitHub pages is a great way to show off your skills as a developer, publish documentation, cheaply host fully custom websites, and more.

In fact, the website you are currently reading is a GitHub pages site (more on this in the next slide). The whole reason this site is different than, let's say, a Google Site, is the fact it fully supports custom HTML, CSS, JS, and whatever else you may want in your site. It's like having a free webserver, but you can link it to you, and your repositories!

Example

A great example of using GitHub pages to make a real, working, site is what your looking at now!

In fact, this whole webpage is being hosted on GitHub pages, even with custom HTML, CSS, JS, and proper Bootstrap! The thing to left, the carousel, is a custom component provided by Bootstrap. The code that makes this text change with the carousel when you click Next → is a custom JS script.

The reason this site works and looks the way it does is proof that GitHub pages is a great alternative to running a standalone web server.
